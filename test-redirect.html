How to Fix Redirect URI Mismatch

If you're seeing a "redirect_uri_mismatch" error, follow these steps:

  1. Go to the Google Cloud Console Credentials page
  2. Find your OAuth 2.0 Client ID and click on it
  3. Under "Authorized redirect URIs", add:
    http://localhost:3002/api/auth/callback/google
  4. Click "Save"
  5. Wait a few minutes for Google's systems to update
  6. Try signing in again

Verify Your Application is Running

Make sure your Next.js application is running on port 3002:

npm run dev

You should see output indicating that the server is running on port 3002.
